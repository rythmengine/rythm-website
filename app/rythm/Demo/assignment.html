@extends(main) @set(title="Assignment")

Assign a part of document to a Java variable

@assign("foo"){ bar }

Output the variable been assigned: @foo.raw()

Assign a part of document to a final Java variable

@assign("bar", true){ foo }

Output the variable been assigned: @bar.raw()

Note, you can to expand Template Source and Java Source to watch what's behind, specifically you need to expand Java source to check the final object declaration
